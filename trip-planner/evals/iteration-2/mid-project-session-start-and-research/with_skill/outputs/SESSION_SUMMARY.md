### Jordan's Strict Vegetarian Needs (No Fish Sauce, No Bonito Dashi)

**Best Options (100% Guaranteed):**
- **Shojin Ryori (Buddhist):** Shigetsu — entirely plant-based, no fish ever used. Zero ambiguity.
- **Fully Vegan Modern:** Uzu (ramen), Ain Soph Journey (casual) — vegan by design, no hidden fish products

**Good Options (Vegetarian with Advance Notice):**
- **Kaiseki (with Request):** Gion Maruyama, Roan Kikunoi — can provide all-plant-based course if dietary restrictions noted when booking. Standard practice for kaiseki restaurants.
- **Casual Noodles:** Hinode Udon, Omen, Marugame Seimen — can request "vegetable broth only, no bonito" — familiar request in Kyoto, staff accommodating

**Research Finding:** Jordan's strict diet is most elegantly served by either shojin ryori (most authentic, cultural experience) or fully vegan modern restaurants (zero ambiguity, creative). Advance-notice kaiseki bookings work but require explicit communication.
