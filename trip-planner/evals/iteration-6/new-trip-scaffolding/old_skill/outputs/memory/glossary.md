# Glossary

## Currencies & Units
| Term | Meaning |
|------|---------|
| USD | US Dollar (home currency) |
| JPY | Japanese Yen (destination currency) |

## Travel Programs & Booking
| Term | Meaning |
|------|---------|
| United miles | United Airlines frequent flyer miles (balance: 85,000) |
| cpp | Cents per point — measures redemption value (miles or points). Calculated as: (cash price in cents) ÷ (points/miles used) × 100. Minimum acceptable: >1.5 cpp for good value. |
| OTA | Online Travel Agency (e.g., Booking.com, Expedia) |
| Direct booking | Booking through the hotel or airline's own website |
| JR Pass | Japan Rail Pass — may be considered for inter-city travel depending on route |

## Japan Travel
| Term | Meaning |
|------|---------|
| Tatami | Traditional Japanese mat flooring (temples, some ryokans) |
| Onsen | Hot spring bath (common in Japanese accommodations) |
| Zen garden | Japanese rock/sand garden designed for meditation |
| Stroll garden | Large landscaped garden designed for walking through |
| Temple complex | Multiple connected temple buildings in one area |
| Shrine | Shinto religious site (different from temples) |

## Traveler Preferences
| Term | Meaning |
|------|---------|
| Booking philosophy | Cost optimization vs. flexibility vs. balanced. Established in Phase 1. |
| Booking philosophy: Balanced | Prefer flexibility (refundable, free cancellation) without overpaying. Willing to use miles if value is good (>1.5 cpp). |
| Comfort planning for Alex | Design around comfort (elevator hotels, fewer stairs, natural rest points) not explicit accessibility. |
| Language support | Materials include detailed restaurant translations and transit instructions to address Alex's language anxiety. |

## Trip Phases
| Phase | What It Is |
|-------|-----------|
| Phase 1: Define | Establish travelers, dates, budget, constraints, preferences (THIS SESSION) |
| Phase 2: Route | Decide cities, order, night allocation |
| Phase 3: Transport | Book flights, trains, transfers |
| Phase 4: Accommodation | Book hotels per city |
| Phase 5: Activities | Plan day-by-day activities per city |
| Phase 6: Restaurants | Research and book restaurants |
| Phase 7: Pre-Trip Prep | Create itinerary guides, packing lists, materials |
| Phase 8: Final Check | Verify all bookings, reconcile budget, final prep |
