# Glossary

Japan trip shorthand and terms.

## Trip-Specific

| Term | Meaning |
|------|---------|
| **Tanisha** | Primary planner, vegetarian |
| **Alex** | Travel companion, omnivore, first-time Asia |
| **Knee** | Alex's soccer injury constraint — stairs cumulative fatigue. Plan for natural rest points. |
| **Language barrier** | Alex's anxiety about first time in Asia, unfamiliar language. Will need clear visual guides + reassurance. |
| **Food-focus** | Both travelers prioritize eating well; both interested in temples and gardens. |
| **Depth trip** | 2-3 cities, 3+ nights each. Immersion over rushing. |

## General Planning

| Term | Meaning |
|------|---------|
| **Phase 1 (Define)** | Establish travelers, budget, constraints, trip style. ← **Current phase** |
| **Phase 2 (Route)** | Research destinations, decide 2-3 cities, create itinerary skeleton. → **Next** |
| **Phase 3 (Transport)** | Book flights, trains, transfers |
| **Phase 4 (Accommodation)** | Research and book hotels |
| **Phase 5 (Activities)** | Plan day-by-day activities, schedule tickets |
| **Phase 6 (Restaurants)** | Research and plan meals, make reservations |
| **Phase 7 (Pre-Trip Prep)** | Create itinerary guides, packing lists, language sheets, daily cards |
| **Phase 8 (Final Check)** | Verify all bookings, confirm logistics, handle last-minute changes |
| **CPP** | Cents per point — measures value of loyalty points redemptions. For United miles, typically 1-2 cents depending on route/cabin. |
| **OTA** | Online Travel Agency (Booking.com, Expedia, etc.). Direct booking (airline/hotel site) often preferred. |

## Airlines & Loyalty

| Term | Meaning |
|------|---------|
| **United miles** | Tanisha and Alex's loyalty currency (85,000 available). Can use toward flights. |
| **Premium Cabin** | Business/First class — higher value per point. |
| **Economy** | Standard seating — lower value per point but good for short/medium hauls. |
| **Fuel surcharge** | Extra cost on award bookings (not always applied on United, depends on route). |

## Japan-Specific (to be expanded Phase 2)

| Term | Meaning |
|------|---------|
| **JR** | Japan Railways — major train operator |
| **JR Pass** | Multi-day rail pass (may or may not be worth it depending on routing; typically good for multi-city trips) |
| **Suica/Pasmo** | Prepaid transit cards (bus, train, subway in major cities) |
| **Convenience store** | 7-Eleven, FamilyMart, etc. — ubiquitous, sell food, drinks, etc. |
| **Onsen** | Hot spring bath — common in Japanese inns and public facilities |
| **Ryokan** | Traditional Japanese inn (often include meals) |

## Accommodation & Dining

| Term | Meaning |
|------|---------|
| **Elevator priority** | Alex's knee means elevator access or ground/low floor preferred when possible. Not a hard rule — comfort consideration. |
| **Vegetarian-friendly** | Tanisha's dietary need. Japan has strong vegetarian tradition — should not be a constraint, but requires planning. |
| **Omnivore options** | Alex eats everything — restaurants should have meat/seafood options available. |
| **Tasting menu** | Multi-course set menu — useful for indicating dietary needs upfront (chef prepares vegetarian/omnivore versions). |

## Document Conventions

| Term | Meaning |
|------|---------|
| **Memory file** | Planning document (on-disk reference, not chat). Prefix: `memory/` |
| **Session log** | End-of-session summary (decisions, changes, next steps). Location: `memory/sessions/session-log-{date}-{letter}.md` |
| **Research file** | Comparative analysis (options, ratings, trade-offs). Location: `memory/research/{topic}-research-{city}.md` |
| **Guide file** | Curated, user-approved recommendations (activities, restaurants). Location: `memory/guides/{city}-{type}.md` |
| **Traveler materials** | Printable outputs (itinerary guide, daily cards, packing lists). Location: `outputs/` |
