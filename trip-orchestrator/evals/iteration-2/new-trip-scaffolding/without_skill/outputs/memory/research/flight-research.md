## Booking Scenarios

### Scenario A: All 85K Miles
- **Route:** USA hub (ORD, LAX, DEN, SFO) → Tokyo (HND/NRT) or Osaka (KIX) → return.
- **Cabin:** Economy.
- **Cost:** 85,000 miles + fuel surcharge (~$150–250).

- **Book:** Economy award (50K–60K miles) + cash upgrade to premium economy (~$400–600).

- **Pros:** More comfort, same miles outlay, lower ticket cost.

- **Cons:** Higher trip cost, ties up $1,200 of $8K budget (25%).

## Routing Options

### Open-Jaw (Preferred)
- **Outbound:** USA hub → Tokyo or Osaka (KIX).
- **Return:** Alternate city → USA hub.
- **Example:** SFO → Tokyo (HND) → ... → Osaka (KIX) → SFO.
- **Advantage:** No backtracking. Osaka (KIX) is convenient return point from Kyoto.
- **Award cost:** Typically same as round-trip Tokyo; sometimes slightly more.

### Round-Trip
- **Outbound:** USA hub → Tokyo/Osaka.
- **Return:** Same city → USA hub.
- **Example:** LAX → Tokyo (HND) → ... → Tokyo (HND) → LAX.
- **Disadvantage:** Requires return to Tokyo (time/money if based in Kyoto/Osaka).
