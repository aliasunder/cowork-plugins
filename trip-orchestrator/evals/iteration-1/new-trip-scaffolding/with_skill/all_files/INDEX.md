## Session Workflow

**At session start:**
1. Read CLAUDE.md (this file's status and conventions)
2. Read TASKS.md (what to work on)
3. Read the latest session log (CLAUDE.md points to it)
4. Understand the travelers (read their profiles)

**During session:**
1. Work on the top item in TASKS.md "Active" or "Up Next"
2. Create/update files as needed
3. Track what's been done

**At session end:**
1. Write a session log to memory/sessions/session-log-{date}-{letter}.md
2. Update CLAUDE.md with new phase (if changed), last session pointer
3. Move completed tasks in TASKS.md to "Done" section (with date and strikethrough)
4. Commit changes if using version control
